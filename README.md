# BurnedBridge
A VK-to-Discord one way relay system using LongPoll and vk_api

### Supports
[x] Name, Surname
[ ] Community (bots)
[x] Messages
[ ] Voice messages
[ ] Stickers
[x] Image attachments
[ ] Videos
[ ] Music
[ ] Polls

Credit to [Alekse—ó Kouznetsov](https://vk.com/id194861150) for the [LongPoll implementation](https://github.com/Alex1249/VkLongPoll-for-user).
